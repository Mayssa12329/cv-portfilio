Personal Project

Position: Master's Degree

Duration: 1 month, 2023

Location: Sfax, Tunisia

Project Overview:

In this personal project, I developed a fully functional website for a Camping Van rental service. The website allows users to register, log in, and book camping vans for their trips. I integrated an intuitive UI/UX design and ensured the website's functionality was seamless.

Key Responsibilities:

  • Developed the front-end and back-end of the website using HTML, CSS, JavaScript, and PHP
  • Implemented a secure user authentication system with registration and login features
  • Created a user-friendly booking system for managing reservations
  • Managed the website's database using PhpMyAdmin for storing user information and bookings

Tools & Technologies:

  • HTML, CSS, JavaScript, PHP
  • Database management with PhpMyAdmin
