All-Terrain Robot Design

Duration: 2022-2023

Location: Sfax, Tunisia

Project Overview:

Engineered a competition-ready all-terrain robot using advanced electrical, electronic, and programming techniques. The robot demonstrated superior performance in stability, obstacle detection, and terrain adaptability.

Key Responsibilities:

  • Designed and implemented circuits using Proteus 8 for precise simulation and design validation.
  • Programmed the robot's functionality using C++ in Arduino IDE.
  • Integrated ultrasonic sensors for obstacle detection and implemented an accelerometer and gyroscope for stability control.
  • Conducted extensive testing and calibration to optimize system performance.

Tools & Technologies Used:

  • Arduino IDE
  • Proteus 8 (Circuit Design)
  • Ultrasonic Sensors
  • Accelerometer & Gyroscope
